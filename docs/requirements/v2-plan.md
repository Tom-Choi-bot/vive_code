# 댕케어 캘린더 V2 기획서 (MVP/V2 Plan)

## 1. 핵심 사용자 (Core Users)
* 노령견/묘 또는 만성 질환이 있는 반려동물을 둔 보호자
* 기존의 단순 루틴 체크리스트를 넘어, 건강 트래킹과 추억(일기)까지 하나의 앱에서 모두 관리하고자 하는 사용자

## 2. 문제 정의 (Problems)
* **통계 부재**: 매일 루틴을 체크하지만, 주간/월간 달성률을 한눈에 볼 수 없어 성취감을 느끼거나 부족한 점을 파악하기 어려움
* **건강 기록 파편화**: 체중 변화나 병원 진료 기록, 예방접종 이력 등을 카카오톡 내게쓰기나 별도의 메모장 앱에 흩어져서 관리하고 있음
* **추억 기록의 한계**: 매일의 돌봄 기록(캘린더)에 간단한 그 날의 사진이나 메모를 남길 수 없어 아쉬움

## 3. 사용자 니즈 (Needs)
* **시각적 편안함**: 이른 아침이나 늦은 밤에 루틴을 체크할 때 눈이 부시지 않도록 다크 모드 지원
* **직관적인 성과 확인**: 한 달 동안 얼마나 루틴을 잘 지켰는지 보여주는 차트(통계)
* **통합 관리**: 체크리스트 앱 안에서 몸무게 추이와 특별한 이벤트(병원 방문 등)를 함께 기록
* **프라이버시 및 속도**: 서버 가입이나 인터넷 연결 없이 오프라인(Local Storage)에서 매우 빠르고 안전하게 동작

## 4. V2 개발 범위 (V2 Scope)
1. **다크 모드**: 시스템 테마 연동 및 수동 토글 지원 (우선 완료됨)
2. **통계 및 리포트**: 잔디심기 또는 막대 차트를 활용한 달성률 시각화
3. **건강 트래커**: 체중 기록 그래프 및 건강 관련 이벤트 태그 기능
4. **다이어리**: 특정 캘린더 날짜에 사진(리사이징 후 Base64 압축)과 메모 기록 기능
