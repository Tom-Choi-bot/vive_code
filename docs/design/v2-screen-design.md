# 댕케어 캘린더 V2 화면 구성 설계안

작성된 PRD(V2 스펙 포함)를 바탕으로 각 화면별 구성과 주요 기능을 표 형태로 정리한 설계안입니다.

| 화면번호 | 화면 이름 | 주요 기능 | 추가 설명 |
| :---: | --- | --- | --- |
| **01** | **Splash (스플래시)** | 앱 로고 노출 및 초기 로딩 | 빠른 데이터 로드 후 Home 화면으로 즉시 이동 |
| **02** | **Home (오늘의 돌봄)** | 오늘의 루틴 목록 렌더링, 체크/언체크, 당일 달성률(%) 표시 | 메인 화면. 데스크탑/모바일 뷰 전환 및 다크모드 토글 버튼 위치 |
| **03** | **Pet Management<br>(반려동물 관리)** | 반려동물 프로필(이름, 사진, 종, 생일 등) 등록·수정·삭제 | 프로필 사진은 리사이징 후 Base64로 압축하여 로컬 스토리지 저장 |
| **04** | **Routine Management<br>(루틴 관리)** | 반복 요일, 하루 수행 횟수, 라벨 색상을 지정하여 루틴 추가·수정·삭제 | 이 화면에서 설정한 규칙에 따라 Home 화면의 체크리스트가 자동 생성됨 |
| **05** | **Calendar<br>(돌봄 캘린더)** | 월간 달력 렌더링, 과거 특정 일자의 루틴 달성 기록 조회 및 수정 | 특정 날짜 클릭 시 해당 일자의 루틴 기록이 하단(그리드 뷰)에 표시됨 |
| **06** | **Health Tracker<br>(건강 트래커 - V2)** | 반려동물 체중 입력 및 꺾은선 차트 시각화, 병원 방문 등 건강 이벤트 기록 | V2 신규 화면. 반려동물 관리 탭 내부 또는 독립된 메뉴로 배치 가능 |
| **07** | **Diary<br>(추억 다이어리 - V2)** | 캘린더의 특정 날짜를 선택해 사진 1장과 텍스트 메모(일기) 기록 | V2 신규 화면. 캘린더 화면과 연동되어 과거 추억을 쉽게 열람 가능 |
| **08** | **Statistics<br>(통계/리포트 - V2)** | 주간/월간 루틴 달성률 분석 (잔디 심기 또는 막대 차트 표출) | V2 신규 화면. 자주 놓치는 루틴을 분석하여 성취도를 높이는 데 기여 |
